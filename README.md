# url-diff

diff two urls by:
1. hostname
2. path
3. query parameters

We primarily use this for debugging VAST requests to make it easy to see the difference when macros and such are compiled

## Installation

download the binary and add it to your `$PATH` from the releases page

## Usage

`url-diff help`
